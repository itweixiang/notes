### docker安装

linux部署需要安装erlang，太复杂了，直接用docker启动吧

```sh
docker run --name=rabbitmq \
-p 5672:5672 \
-p 15672:15672 \
-e RABBITMQ_DEFAULT_USER=admin \
-e RABBITMQ_DEFAULT_PASS=weixiang@9104 \
-v "/opt/rabbitmq/data":"/var/lib/rabbitmq"
--restart=always \
-d rabbitmq:3-management
```
